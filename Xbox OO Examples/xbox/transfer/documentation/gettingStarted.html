Getting Started

Getting started with Transfer is relatively straight forward, and there are a lot of resources available to get you going.

What you need

  • Download Transfer.
  • A ColdFusion editor, such as:
    • Eclipse, with the CFEclipse plugin
    • Dreamweaver
    • Homesite
  • An XML editor, such as:
    • For Eclipse:
      • Web Tools Project
      • XMLBuddy
      • oXygen
    • XmlSpy
    • CookTop

Presentations

There have been some presentations made on Transfer, and some have been recorded on Adobe Connect. They are a great place to get an overview of what Transfer does, and how it works.

  • Developing Applications with Transfer ORM - Recorded via Connect (06 November 2006).
    This presentation is based on the 0.6 version of Transfer.
  • Transfer ORM - Presentation Slides (22 June 2006).
    This presentation is based on the 0.5 version of Transfer, but is still quite relevant. Please note that this contains a sample application that will not work with v0.6 of Transfer without some modification.
  • Mark Mandel talks about his Transfer ORM Framework (20 February 2007).
    An audio interview with Nick Tong where we talk about Transfer, and frameworks in general.

All presentations are available here

Tutorials

There are a few tutorials available on Transfer, and they are also a great place to get your feet wet when starting to get going with Transfer. 

General 

  • Doing CRUD Operations in Transfer
    This goes over the basics of doing simple Create, Read, Update and Delete operations with Transfer, from the ground up.
  • Primary Key Control with Transfer
    This looks at the specific mechanisms that are available with Transfer for handling primary keys.
  • Building an Application with Transfer ORM (Brian Rinaldi)
    This is another very good introduction to Transfer, and looks at configuring Transfer to aid in the development of the CF Open Source List
  • Baby Steps with Transfer (Damon Gentry)
    A great introduction to Transfer, Damon chronicles his learning of Transfer as he builds his application.

Framework Specific

  • Setting up and using Transfer within fusebox via the Transfer lexicons (Nick Tong)
    Setting up Transfer within fusebox via the Transfer lexicons

Example Code

There are several example codebases that are available to see how Transfer works in a real application.

  1. tBlog
    This is a simple sample blog application. Its configuration file is heavily commented, and is a great place to start from when learning Transfer.
  2. tCalendar
    This is a small Calendar application, that is also useful to see how composite elements work together.
