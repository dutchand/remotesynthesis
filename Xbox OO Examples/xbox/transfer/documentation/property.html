Default Property Values

By default, Transfer sets values to properties that are found on TransferObjects.

These default values are:

  • string: ""
  • numeric: 0
  • boolean: false
  • date: Now()
  • UUID: a random UUID
  • GUID: a random GUID

To see more on how to set your own defaults, see the 'configure' method in the Custom Methods or Using Decorators sections of the documentation.
