Supported Systems

This is currently just a list of tested ColdFusion versions and Databases. In theory Transfer should work with more, but they have yet to be tested at this time.

ColdFusion Versions:

  • ColdFusion MX 7

Database Servers:

  • MS SQL Server 2000+
  • MySQL 4.1+
  • PostGres 8.1+
  • Oracle 9i+

Not Supported

ColdFusion Versions:

  • ColdFusion MX 5 and below

Database Servers:

  • MS Access (any version)
