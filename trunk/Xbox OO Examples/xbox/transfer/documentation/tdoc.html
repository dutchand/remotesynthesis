TransferDoc

TransferDoc is a utility to document specific TransferObject configurations within your application, without forcing you to reread the Generated Methods documentation repeatedly.

Installation

To install transfer doc, download it from the downloads page, and unzip it into the direcory of the application that is utilising Transfer.

Create the TransferFactory

Opening the page for TransferDoc, to use the application to generate documentation, you will first need to Fill in the form above the '(Re)Create the TransferFactory' to pass in relative paths to the appropriate XML files, and definitions folder.

This will probably be the same as the code used to initialise the TransferFactory in your application.

Once the TransferFactory has been initialised, the line 'TransferFactory has been initialised' will display, and link to the Transfer configuration file will display.

Generate Documentation

To generate documentation, select the class that you wish to generate documentation for in the box next to the label 'Transfer Class', and click the 'Generate Documentation' button.

The documentation will now display on the page.

Generated Documentation

Here you will find the list of documentation that has already been created.

Clicking on the link to the document will open the document in the page.

Clicking 'refresh' will reload the TransferObject and recreate the documentation.

Clicking 'delete' will delete the documentation.

Clicking 'FlashPaper' will open the documentation directly from the FlashPaper SWF.
